# FlowOps AI 方針 / AIMS 運用ルール（正本）

> AIMS 管理文書（ISO/IEC 42001 5.2 / A.2）｜状態: `active`｜更新: 2026-06-20｜次回: 2026-12-20（半期）
> 承認: PO（マネジメントレビュー）。改訂は PR＋AuditLog(`DOC_UPDATE`)。

## 1. 適用範囲 (4.3)

- 対象: Intake / Proposal生成 / リスク・影響評価アシスト / Agent説明
- データ: `spec/**` / Issue・Proposal・Evidence / 監査ログ / LLM入出力
- 対象外: ロボット実機制御 / 最終意思決定の自動化 / 本番電子署名・法規制対応
- **AI は助言者であり意思決定者ではない**

## 2. 方針声明 (5.2 / A.2.2)

1. 人間中心 — 最終承認は常に人
2. 追跡可能性 — 誰が・いつ・何を・なぜを監査ログに残す
3. 可逆性 — 変更は Git 版管理、差し戻し可能
4. 比例性 — 管理策はリスクに比例
5. 継続的改善 — 差し戻し・不適合は改善の入力

上位方針と整合(A.2.3)し半期レビュー(A.2.4/9.3)。

## 3. 運用ルール（DecisionOps 5ルールの AIMS 再定義）

正本は [decisionops-operating-model.md](../decisionops-operating-model.md)。判定基準のみ示す。

| Rule | 規範 | 判定基準 | 対応 |
|---|---|---|---|
| 1 | LLM出力=提案 / Git=正本 | 人レビューなしに `active` 反映していない | A.6.2/A.7.5 |
| 2 | `active` だけが本番 | ライフサイクル付与＋`active`遷移に人承認記録 | A.9.2 |
| 3 | 計算式は version 保持 | gateId/gateVersion＋前提を GateEvaluation・AuditLog に二重記録 | A.6.2.8/8.2 |
| 4 | 可変層は現場裁量 | 可変層の変更が普遍層の承認・証跡を迂回しない | 6.1/A.5.4 |
| 5 | 差し戻しは資産 | 差し戻し率・理由を週次集計し再発防止に接続 | 10.1/10.2 |

普遍層=価値ノード構造/受入ゲート/監査ログ/変更理由/人承認。可変層=KPI/閾値/危険源/センサ/PLC・DCS/現場ルール。

## 4. 役割と責任 (5.3 / A.3.2)

| 役割 | 責務 |
|---|---|
| PO | 方針承認、目標・受容基準決定、マネジメントレビュー主催 |
| テックリード | 管理策(ゲート/ライフサイクル/CI)の実装・維持 |
| 改善担当(持回り) | 監視集計、内部監査、不適合追跡 |
| 起票者 | リスク・影響・事象の起票、証跡提供 |
| 承認者 | Decision Card での監督・承認・差し戻し |

**懸念通報(A.3.3)**: `aims:nonconformity` Issue または改善担当へ、報復なく通報可。

## 5. AIMS 目標 (6.2)

| 目標 | 指標 | 目標値 | 集計元 |
|---|---|---|---|
| 人間監督 | `active`前の人承認率 | 100% | AuditLog |
| 追跡可能性 | 主要AI操作のログ欠損 | 0 | 監査整合チェック |
| 提案品質 | 提案採択率 | >60% | Proposal状態 |
| 是正俊敏性 | 不適合 起票→完了 中央値 | <10営業日 | Issue状態 |
| リスク低減 | 高リスクで影響評価実施 | 100% | 影響評価/ゲート |

## 6. リスク・影響 (6.1 / 8.2-8.4)

手順は [risk-and-impact-assessment.md](risk-and-impact-assessment.md)。高リスク（人身安全・個人/集団への重大影響・法規制）は影響評価必須、安全レビューゲート通過まで `active` 化しない。

## 7. 文書・記録 (7.5 / A.4.2)

`docs/aims/**`・`spec/aims/**` は管理文書（変更は PR＋AuditLog）。監査ログは不変（UPDATE/DELETE禁止）。

## 8. 違反時

逸脱を発見したら `aims:nonconformity` を起票し是正フロー（[手順 §5](risk-and-impact-assessment.md)）に従う。放置して `active` 反映しない。
